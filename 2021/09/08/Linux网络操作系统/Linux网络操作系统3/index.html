Linux网络操作系统3

一、Linux运行级别

1.查看当前系统的运行级别。
[root@localhost-]#runlevel

N 5
2.查看开机默认的运行级别。
[root@localhost-1#systemctl get-default

graphical.target

3.设置默认的运行级别为命令行模式。

[root@localhost-]#systemctl set-default multi-user target
[root@localhost-]#systemctl get-default multi-user.targe

4.切换当前运行级别到命令行模式。

[root@localhost-]#systemctl isolate multi-user.target

或者
[root@localhost-]#systemctlisolate runlevel3.target

二、Linux操作系统基本操作

命令 含义
#halt 关机
#shutdown （-h-r）now
#reboot 重起
#root 登陆
#rlogin 远程登录
#login 注销
#startx 进入图形界面
#pwd 查看当前所处的目录的绝对路径（我在哪里）
#ls
#cd

三.Linux操作系统基本操作

命令 含义 备注
root 超级用户目录 包括桌面管理等
home 用户目录 包括用户信息等
bin 执行目录 存放可执行文件包括常用命令
sbin 执行目录 存放不提供给用户使用的命令
boot 引导目录 存放引导系统时使用的各种文件
etc 配置目录 系统配置时使用
dev 设备目录 通过它访问外设
mnt 安装目录 管理员设备临时安装点
opt 安装目录 管理员软件包放置点
lib 库目录 命令执行时使用
usr 共享目录 存放系统中所有用户的共享文件
var 数据目录 系统运行时要修改数据
tmp 临时目录
proc 虚拟目录 虚构文件系统内存产生不存硬盘
lost+fount 空目录

文件类型颜色
一般文件————-白色
目录文件————-蓝色
符号链接文件——-紫色
设备文件————–黄色
可执行文件———–青绿色
图片文件————–粉红色
压缩文件————–红色
